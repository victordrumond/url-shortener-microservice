URL Shortener

Usage

Make a POST request above with a valid URL address and get a shortened URL.

Then, you can make a GET request to /api/shorturl/:shortened-url? and you will be redirected to the original URL.

/api/shorturl/8SHzTdN03

Output

{"original_url": "https://www.google.com/", "short_url": "8SHzTdN03"}
